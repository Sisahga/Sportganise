# Backend

A wonderful java app.

## To run the application

Unix systems

```bash
./mvnw spring-boot:run
```

Windows

```powershell
./mvnw.cmd sprint-boot:run
```

## To run the test suite

Unix systems

```bash
./mvnw test
```

```powershell
./mvnw.cmd test
```

## To lint the project

```bash
./mvnw stylecheck:check
```
